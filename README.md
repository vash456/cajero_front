
# Cajero - Aplicación de Gestión Bancaria

## Descripción

Aplicación web frontend de un sistema de cajero automático que permite a los usuarios gestionar sus cuentas bancarias de forma segura.

## Características

- **Autenticación de Usuarios**
    - Registro de nuevas cuentas
    - Login seguro

- **Gestión de Cuenta**
    - Consulta de saldo disponible
    - Historial de transacciones

- **Movimientos Bancarios**
    - Retiro de efectivo
    - Depósito de fondos
    - Transferencias entre cuentas

## Tecnologías

- HTML5
- JavaScript

## Instalación

```bash
# Clonar el repositorio
git clone https://github.com/usuario/cajero_front.git

# Navegar al directorio
cd cajero_front

# Abrir en navegador
open index.html
```

## Uso

1. Registrarse con email y contraseña
2. Iniciar sesión con credenciales
3. Consultar saldo e historial
4. Realizar movimientos bancarios
5. "Limpiar datos" resetea todos los registros.

Nota: se tiene por defecto un usuario administrador (username="admin", password="1234")
